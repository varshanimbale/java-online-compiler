# 🚀 Java Online Compiler

A full-stack Java Online Compiler built using Spring Boot and Monaco Editor.

## 🔥 Features
- 📝 Code Editor (VS Code style)
- ▶ Compile & Run Java code
- ⌨ Input support (Scanner)
- 📤 Real-time output display
- ⚡ Fast execution

## 🛠 Tech Stack
- Java 21
- Spring Boot
- HTML, CSS, JavaScript
- Monaco Editor
